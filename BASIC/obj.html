JS Objects

keyword : objek js

  • kamu udah tau bahwa variabel javascript = wadah untuk nilai data
  • syntax dibawah untuk memberikan nilai "harry" ke variabel mobil
    • var car = "harry";
  • object juga variabel, bedanya bisa menyimpan lebih byk nilai
  • syntax dibawah untuk memberikan banyak nilai "harry", "500", "white" ke obj mobil
    • var car = {
    name:"harry",
    model:"500",
    color:"white"
    };

Definisikan Object

  • buat object dgn obj literal
  • name, age, eyeColor = property
  • Shiro, 30, blue = value
  • sayHello = method/function

    • var human = {
    name: "Shiro",
    age: 30,
    eyeColor: "blue",
    sayHello : function() {
    return "I'm" + this.name + "I'd like to meet you";
    //"this" mengacu pada pemilik property/method
    }
    }

Akses Propery Object

  1. cara 1 => objectName.propertyName

    2. human.eyeColor = "yellow"; //ganti value
    alert(human.name); //tampilkan value

  2. cara 2 => objectName["propertyName"]

    3. human["name"] = "Aries"; //ganti value
    alert(human["name"]); //tampilkan value
