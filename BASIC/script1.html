Cara-cara Menambah/Menghubungkan JavaScript

keyword : konek js

  1. Eksternal js

    2. disarankan ditaruh sebelum tag penutup body, boleh didlm tag head/body juga

    <!DOCTYPE html>
    <html>
    <head>
    </head>
    <body>
    <h1>This is a heading</h1>

    <script src="myscript.js"> <--- INI NEHH

    </body>
    </html>

  2. Internal js

    3. ditaruh sebelum tag penutup body, boleh didlm tag head/body juga

    <!DOCTYPE html>
    <html>
    <head>
    </head>
    <body>
    <p>This is a paragraph.</p>

    <script> <--- MULAI DARI OPEN TAG GEYS
    console.log('hello !'); </script>     <--- SAMPE CLOSE TAG SINI

    </body>
    </html>

  3. Inline js

    4. langsung taruh js functionnya beserta nilainya diobjek target

    <!DOCTYPE html>
    <html>
    <body>

    <button onclick="alert('hello world')">klik saya</button>

    </body>
    </body>
    </html>
