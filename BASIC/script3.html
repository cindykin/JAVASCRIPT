Macam-macam keluaran JavaScript

keyword : output js

  1. Merubah isi property elemen HTML, pakai cara DOM

    2. Merubah property innerHTML dari sebuah elemen HTML is cara umum utk menampilkan data di HTML

    <!DOCTYPE html>
    <html>
    <head>
    </head>
    <body>
    <h1 id="hehe"></h1>

    <script>
    document.getElementById("hehe").innerHTML = "eyo gais";
    </script>

    </body>
    </html>

  2. Keluaran ke HTML, pakai document.write()

    3. Menggunakan document.write() setelah dokumen HTML dimuat, akan menghapus semua HTML yang ada (Never call document.write after the document has finished loading. It will overwrite the whole document)

    <body>
    <h1 id="hehe"></h1>

    <script>
    document.write('eyo gais');
    </script>

    </body>


  3. Pakai window.alert()

    4. menggunakan kotak peringatan untuk menampilkan data

    <body>
    <h1 id="hehe"></h1>

    <script>
    alert('hey winner');
    </script>

    </body>


  4. Pakai console.log()

    5. Untuk tujuan debugging, nanti hasil akan muncul diconsole browser

    <body>
    <h1 id="hehe"></h1>

    <script>
    console.log('thank you for visiting');
    </script>

    </body>
