Tipe Data JavaScript

keyword : tipe data js

konsep number & string js

Variabel JavaScript dapat menampung byk tipe data spt angka, string, objek, dll.
Saat menambahkan angka dan string, JavaScript akan memperlakukan angka tersebut sebagai string

  • number duluan
    • var x = 16 + 4 + "Volvo";

    JavaScript memperlakukan 16 dan 4 sebagai angka, hingga mencapai "Volvo"
    hasil => 20Volvo

  • string duluan
    • var x = "Volvo" + 16 + 4;

    karena operan pertama adalah string, semua operan diperlakukan sebagai string
    hasil => Volvo164

konsep timpa variabel js

JavaScript memiliki tipe data dinamis. Ini berarti nama var yang sama dapat digunakan untuk menampung tipe data yang berbeda

        var x;           // Now x is undefined
        x = 5;           // Now x is a Number
        x = "John";      // Now x is a String

  1. Strings

    2. ditulis dengan tanda kutip ganda/tunggal

    var carName1 = "Volvo XC60"; // Using double quotes
    var carName2 = 'Volvo XC60'; // Using single quotes

  2. Numbers

    3. jika dlm desimal tambahkan titik

    var x1 = 34.00; // Written with decimals
    var x2 = 34; // Written without decimals

  3. Booleans

    4. hanya ada 2 keluaran = true & false

                    var x = 5;
                var y = 5;
                var z = 6;
                (x == y)       // Returns true
                (x == z)       // Returns false

  4. Arrays

    5. ditulis didalam [ ], dipisah dgn koma(,) , perhitungan item dimulai dari 0

    var cars = ["Saab", "Volvo", "BMW"];

  5. Arrays

    6. ditulis didalam { }, ditulis dgn pasangan nama: nilai, dipisah dgn koma(,) , perhitungan item dimulai dari 0

    var person = {firstName:"John", lastName:"Doe", age:50, eyeColor:"blue"};

  6. Identifikasi Tipe Data

    7. mengembalikan apa tipe data yg dimaksud

                    typeof ""             // Returns "string"
                typeof "John Doe"     // Returns "string"
                typeof 0              // Returns "number"
                typeof 3.14           // Returns "number"
                typeof (3)            // Returns "number"
                typeof (3 + 4)        // Returns "number"

  7. Undefined

    8. variabel tanpa nilai, memiliki nilai undefined. Jenisnya juga undefined

    var car; // Value is undefined, type is undefined

  8. Empty Values

    9. Nilai kosong tidak ada hubungannya dengan undefined

    var car = ""; // The value is "", the typeof is "string"

  9. Null

    10. null js adalah "tidak ada". Itu seharusnya menjadi sesuatu yang tidak ada. Sayangnya, dalam JavaScript, tipe datanya null adalah sebuah objek.

    var person = {firstName:"John", lastName:"Doe", age:50, eyeColor:"blue"};
    person = null; // Now nilainya = null, tp tipe datanya masih sebuah obj

    var person = {firstName:"John", lastName:"Doe", age:50, eyeColor:"blue"};
    person = undefined; // Now baik nilai & tipenya = undefined

    Perbedaan Undefined dan Null

    undefined dan null memiliki nilai yang sama tetapi jenisnya berbeda

                    typeof undefined           // undefined
                typeof null                // object

                null === undefined         // false
                null == undefined          // true
